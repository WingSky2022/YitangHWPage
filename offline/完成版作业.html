我的作业
1、初阶作业：学习心得题
课上，我们用一个有趣的STAR模型框架，系统地见识了、理解了一波AI落地场景，了解背后的机制。

你之前对AI的使用落地是怎么理解的，听完课之后，有什么变化吗？欢迎写一写你的新认知和新思考。

之前对AI落地主要是基于需求，如果没有需求场景，很难单纯为了兴趣或能力提升来学AI。

即便是以未来3到5年来进行练习，依旧需要一些适当的反馈。

最好的学习是有落地场景，这也符合刻意练习中「及时反馈」这个要素。


学完课后，发现自己对AI落地的场景之前只停留在了「提速场景」和「探索场景」上，对于AI能否真正带来利润提升的「落地场景」和AI可能带来的「变革场景」，还缺乏这方面的强意识。

将STAR记在心里，碰到思考AI落地场景，就拿这个图来思考下，也许渐渐能有新得。

2、进阶作业：场景收集题
过去，你在生活中、自己的业务or所在的团队，已经有了哪些使用AI的场景？
请你结合提速、探索、落地、变革的四象限框架，多提一些假设，做一个AI场景的加法池子。最好是用清单体笔记的方式。

然后再给这些场景假设做一个排序，挑选出你最想实现的优先级top3。

提速：AI的使用解决了某些痛点，如果回退到过去不用AI的场景，会很痛苦。

- 会议纪要/笔记整理/知识萃取 # 已实现

- 生活指导：电脑系统重装（省了两百块），换净水滤芯（省了三百块） # 已实现

- 脚本自动分镜

- 智能版权核查

- 智能素材收集整理系统 # Top1



探索：探索自己能力边界，在两三年前甚至不敢相信自己能做到这样的事。

- VibeCoding # Top2

- 深度学习

- AI绘图、AI生视频

- AI写歌

- 情绪可视化：通过AI将素材内容呈现的情绪整理为可视化符号

- 多风格融合实验


落地：大幅提升效率/收入/利润，很难回退，并且激励团队使用AI

- 品牌AI短片定制化

- AI短视频课程产品

- IP孵化工厂：批量创作故事，通过数据筛选有商业价值的IP


变革：从底层的公司基础设施、HR制度、团队分工，完全AI化融入，打造一个人和AI高效协同的新公司。

- AI制片厂：重构传统影视行业，AI承担八成以上的输出

- 个性化内容引擎：用户可自定义生成符合自己喜好的内容 # Top3

3、高阶作业：未来展望题
请发挥你的想象力，给自己想做的事情许个愿吧。在未来的3~10年，由于AI场景成为现实，会给你自己、你的业务带来什么样的变化？想要达成这个美好的愿景，你的目标是什么，怎样拆解？

这里是帮你练习美好想象力、目标管理和里程碑拆解，随着科技的浪潮一波一波迭代，这些愿景实现的速度，可能比我们预期的更快。

这是一个非常有趣的作业，欢迎大家可以挑战一下啊~~

1. 愿景：成为一个能创作并发布高质量AI短片的「造梦师」。

2. 里程碑拆解

1. 2025年：掌握AI短视频制作全流程，建立个人审美和素材子弹库。

2. 2026年：形成个人独特叙事风格，发布AI剧集。

3. 2027年：创作具有影响力的作品。

4. 2028年：构建个人创作生态，探索更多的叙事媒介。

3. 基本功拆解

1. 故事内核能力：故事构思、角色塑造、情节设计能力修炼

2. AI多模态技术力：熟练使用多模态工具，并逐步探索对应的工作流工具。

3. 视觉审美能力：迭代最佳实践池子，形成独特视觉审美。

4. 项目管理能力：系统化管理从idea到成本的流程。
