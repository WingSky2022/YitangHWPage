# 作业系统启动说明

## 文件说明

- `index.html`：完整版本（需要网络连接，支持Markdown实时预览）
- `offline.html`：离线版本（无需网络，使用普通文本框）

## 快速启动

### 方法一：直接打开离线版本（推荐，无需网络）
1. **双击文件**：直接打开 `offline.html` 文件
2. **右键打开**：右键点击 `offline.html` → "打开方式" → 选择您的浏览器

### 方法二：打开完整版本（需要网络）
1. **双击文件**：直接打开 `index.html` 文件
2. **右键打开**：右键点击 `index.html` → "打开方式" → 选择您的浏览器

### 方法三：浏览器中打开
1. 打开任意浏览器（Chrome、Firefox、Edge、Safari）
2. 按 `Ctrl+O`（Windows）或 `Cmd+O`（Mac）
3. 选择并打开 `offline.html` 或 `index.html` 文件
4. 或者在地址栏输入完整文件路径，如：
   ```
   file:///F:/GiteePro/CodeUsed/一堂作业优化/GLM/offline.html
   ```

### 方法四：使用本地服务器（推荐开发使用）
如果您有VS Code，可以安装 "Live Server" 插件：
1. 在VS Code中安装 "Live Server" 插件
2. 右键点击 `offline.html` 或 `index.html` → "Open with Live Server"
3. 页面将自动在浏览器中打开并支持热重载

## 功能预览说明

### 已实现的功能
1. **界面切换**
   - 右上角按钮可切换亮色/深色主题
   - 主题设置会自动保存

2. **Markdown编辑**
   - 支持 `# 标题`、`## 二级标题`
   - 支持 `**粗体**`、`*斜体*`
   - 支持有序列表 `1. 项目`
   - 支持无序列表 `- 项目` 或 `* 项目`
   - 支持粘贴Markdown文本自动转换
   - 支持快捷键：`Ctrl+B` 加粗、`Ctrl+I` 斜体

3. **草稿功能**
   - 自动保存（每30秒）
   - 手动保存：点击"暂存草稿"按钮
   - 页面刷新后会提示恢复草稿
   - 草稿7天后自动过期

4. **响应式设计**
   - 手机端自适应
   - 平板端优化
   - 桌面端完美显示

### 常见问题

**Q：为什么样式显示不正确？**
A：请确保使用较新版本的浏览器，或尝试刷新页面（Ctrl+F5）

**Q：为什么主题切换没有效果？**
A：可能是浏览器禁用了localStorage，请检查浏览器设置

**Q：为什么提交按钮没反应？**
A：这是演示版本，提交功能只是模拟，不会真实发送数据

**Q：如何清除保存的数据？**
A：按F12打开开发者工具 → Console（控制台）→ 输入：
```javascript
localStorage.clear()
```
