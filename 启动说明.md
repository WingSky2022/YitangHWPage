# 作业系统本地运行说明

## 问题说明

这些HTML文件是Vue.js单页应用的已编译版本，依赖以下资源：
- CDN上的JavaScript模块文件
- 后端API接口
- 外部样式和字体文件

## 推荐打开方式

### 方式1：使用本地HTTP服务器（推荐）

#### 方法A：使用Python（如果已安装Python）
```bash
# Python 3
cd "F:\GiteePro\CodeUsed\一堂作业优化"
python -m http.server 8000

# 然后在浏览器打开：
# http://localhost:8000/未填写的作业.html
# http://localhost:8000/完成版作业.html
```

#### 方法B：使用Node.js（如果已安装Node.js）
```bash
# 安装http-server（只需安装一次）
npm install -g http-server

# 运行服务器
cd "F:\GiteePro\CodeUsed\一堂作业优化"
http-server -p 8000

# 然后在浏览器打开：
# http://localhost:8000/未填写的作业.html
# http://localhost:8000/完成版作业.html
```

#### 方法C：使用VS Code的Live Server扩展
1. 在VS Code中安装 "Live Server" 扩展
2. 右键点击HTML文件
3. 选择 "Open with Live Server"

### 方式2：直接双击打开（需要网络）

1. **确保网络连接正常** - 需要访问CDN资源
2. **使用现代浏览器** - Chrome、Edge、Firefox等
3. **如果页面空白或闪退**：
   - 按F12打开开发者工具
   - 查看Console（控制台）标签页
   - 检查是否有错误信息

## 常见问题

### Q: 页面打开后空白？
A: 可能是：
- 网络问题，CDN资源加载失败
- Vue应用未初始化
- 查看浏览器控制台的错误信息

### Q: 功能不完整？
A: 这些HTML依赖后端API，如果只是查看样式和布局：
- 可以直接打开查看
- 如果需要完整功能，需要连接原始服务器

### Q: 如何只查看样式效果？
A: 即使没有后端，页面样式、深色模式、Markdown提示等功能应该可以正常显示。
